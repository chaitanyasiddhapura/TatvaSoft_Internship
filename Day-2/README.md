
# Day 2 – Angular 14 Login Form with Bootstrap

## 📌 Task Overview

As part of my 15-day internship at **TatvaSoft**, Day 2 focused on learning the basics of **Angular 14**. The task involved:

- Setting up an Angular 14 project
- Installing and configuring **Bootstrap**
- Creating a simple **login form**
- Printing form data (email & password) in the console

---

## 🚀 How to Run the Project

### ✅ Step 1: Clone or Download the Project

If not already done:
```bash
git clone <your-repo-url>
cd day2-login-app
```
### ✅ Step 2: Install Dependencies
Make sure you have Node.js and Angular CLI installed.
If not:
```bash
npm install -g @angular/cli@14
```
Then install project dependencies:
```bash
npm install
```

### ✅ Step 3: Run the Angular App
```bash
ng serve
```
Open your browser and navigate to:
http://localhost:4200/

---
###  🧪 Features
- Responsive login form using Bootstrap
- Two-way data binding with [(ngModel)]
- Form submission using (ngSubmit)
- Console logs the entered email and password
